Lalalalala....

Written by Covey.Liu with ♥ on February 11, 2017 in

坚持经典 Jazz 的人眼里的世界「既斗争又妥协」，渴望在舞台上光彩照人的人在乎观众的目光首肯，相信罗曼蒂克的还存在的人一定没穿着高跟鞋。

拥有什么样的理想决定了看到什么样的世界。

用 70s 滤镜讲了一个现代故事，用大块的鲜艳颜色重新点缀女性的衣装。这种复古感让歌舞不古怪突兀，歌舞也让让这个越来越冰冷的世界重新温暖浪漫起来。

只是有些遗憾导演没有让歌舞片植根于新的土壤，只是重新修剪旧园那树那叶，况且歌舞元素在后半部略显毛糙。

摄像上唯「流畅」一词可以形容，开场一曲足以叫人叹为观止。

PS.在电影院开着外放看动画片的小朋友，你再这样要被我打的。
