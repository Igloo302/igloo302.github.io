流浪动物共养计划

Written by Igloo with ♥ on October 29, 2018 in

2018

发起人

商业模式设计

该项目利用自动贩售、互联网社交平台等技术，在线下设立自助猫狗粮售卖点和猫狗住所，并在此基础上建立网上流浪动物讨论社群作为公益宣传渠道和志愿者来源，通过实物猫狗粮售卖利润、网上领养流浪动物的“云养猫/狗”游戏化增值服务、鼓励公益捐款等方式引入商业资本并将其投入流浪动物的TNR（Trap Neuter Release 诱捕、绝育、放生）管控和健康服务（免疫、驱虫、救助、义诊）之中。

该项目主张流浪动物的所有权不该属于个人，动物应该与人以一种更平等的方式相处。通过上述商业模式，取代效率低下的收容+领养和人道缺失的扑杀的处理手段，实现流浪动物的人道主义救助和数量管控，加强人与流浪动物的联系，以期建立一种流浪动物和谐共生的崭新的未来城市生活方式。
