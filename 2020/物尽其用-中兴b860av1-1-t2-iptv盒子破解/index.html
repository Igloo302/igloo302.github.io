物尽其用|中兴B860AV1.1-T2 IPTV盒子破解

Written by Igloo with ♥ on March 15, 2020 in

注意：仅在杭州电信B860AV1.1-T2测试。

软破法

所谓软破就是不需要使用特殊的硬件，通过打开B860AV1.1的ADB来修改系统。

  1. 配置网络

    打开盒子，点击遥控器的「设置」，密码为6321。将盒子连接到有线网络或者Wifi（忽略鉴权相关设置）

  2. 打开ADB

    在设置中，找到打开ADB，需要输入密码。下载ADB密码计算器

  3. 通过ADB连接盒子，在电脑上输入命令：

adb connect 192.168.1.xxx
adb shell
su
mount -o remount -o rw /system
  1. 后续的操作和拆机法相同。

拆机法

该方法为通用方法，需要准备硬件USB转TTL线（一般型号CH340G），如果板子上没有焊好针脚的话，还需要点焊枪等工具。

  1. 拆机接线

    2号接GND线，5接TX线，6接RX线，针脚排列见图（图源见水印）

  1. 将USB口连接到电脑，打开设备管理器 - 端口 - COM - 属性 - 端口设置，将波特率改成115200
  2. 打开Putty，连接模式设置为serial（串口），填入设备管理器中的Serial line和Speed，点击Open

  1. 给盒子通电，putty开始跑码，同时插入存有桌面Apk文件的U盘

  2. 输入命令

# 打开adb
start adbd

# 查看U盘路径
df

# 复制桌面App到系统中
cp launcher.apk /data/app

# （如果安装是当贝桌面）打开当贝桌面
am start com.dangbei.tvlauncher

  1. 修改首页键，使其打开当贝桌面

    现在，点击首页键会回到IPTV。需要添加一个脚本，将首页键映射为打开当贝桌面。创建一个init.insmod_usb.sh文件，将其复制到 /etc/文件夹。

#!/system/bin/sh

insmod /system/lib/modules/dwc2.ko
while [ 1 ]
do
	rm_code=`getevent -c 1 /dev/input/event1`        
	if [ "$rm_code" = "0001 00a9 00000000" ] ; then
		$rm_code
		am start -n com.dangbei.tvlauncher/com.dangbei.tvlauncher.IndexActivity
	fi     
done

cp init.insmod_usb.sh  /etc/
chmod 777 /etc/init.insmod_usb.sh
